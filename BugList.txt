1. При создании объекта File при передаче параметра filename без расширения поле extension получает значение filename
2. При создании объекта File при передаче параметра content длиной 1 элемент поле size типа double становится равным 0,5, при попытке получить значение этого поля возвращается 0
3. При создании объекта File при передаче в качестве параметра filename пустой строки объект создается, но не должен
4. Конструктор объекта FileStorage позволяет создать объект с отрицательным или нулевым значением параметра size
5. При создании объекта FileStorage с параметром size поле availableSize получает значение size+100, а должно становиться равным size
6. При создании объекта FileStorage с параметром size, равным Integer.MAX_VALUE+1 и более, поле maxSize принимает значение -size,
а поле availableSize принимает значение -size+100
